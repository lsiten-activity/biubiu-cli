# biubiu-cli

```
Usage: biubiu [options] [command]

Options:
  -v, --version      output the version number
  -h, --help         display help for command

Commands:
  init               初始化 biubiu config 配置文件
  create [template]  生成 biubiu 模板
  release            发布模板
  help [command]     display help for command
```
